## ‚ÜîÔ∏è STF DDS Converter

STF DDS Converter is a lightweight desktop tool designed to convert between `.stf` textures and `.dds` graphics. STF textures are used in **[Operation Flashpoint: Dragon Rising](https://en.wikipedia.org/wiki/Operation_Flashpoint:_Dragon_Rising)**, a tactical military shooter developed by [Codemasters](https://www.codemasters.com). This tool is particularly useful when modding game menus/UI screens and in-game environments. STF DDS Converter features a modern, user-friendly design.

## üì• Installation

**STF DDS Converter** is a portable application - no installer required. Just download the latest build from the **[Releases](https://github.com/NoahDomingues/FUI-SWF-Converter/releases)** section, unzip the archive, and run the executable:

- **Windows**: Double-click `STF DDS Converter.exe` to launch, or run it from your command line.
- **Linux**: Install [WINE](https://linuxconfig.org/installing-wine), and run this command via terminal:  `wine "STF DDS Converter"`

## üí° Usage

<a href="https://discord.gg/Z88NnTgpWU"><img width="100%" alt="image" src="https://github.com/user-attachments/assets/22637f56-ec45-45cd-86d2-f6d761aed62b" /></a>

- Click the **Select.stf File** button to choose a .stf file for conversion
- Click the **Select.stf File** button to choose a .stf file for conversion
- Use the **Convert** button to convert the selected `.stf` file to `.dds` or the selected `.dds` file to `.stf`
- The **progress bar** shows conversion status, and the **log box** displays detailed output.
- If the output file already exists, STF DDS Converter will prompt you to either overwrite the existing file or cancel the operation

Converted files are renamed and saved in-place. The tool does not overwrite original files unless manually confirmed.
